jogador1 e jogador2 enviam jogadas para o servidor. Servidor cuida de toda a parte do jogo de receber
as msg enviadas pelos jogadores e fazer validações. 
Compilar no terminal "servidor.c" -- gcc -Wall servidor.c -l rt
Compilar no terminal "jogador1.c" e "jogador2.c" -- gcc -Wall jogador1.c -o jogador1
Executar primeiro o servidor e em outro terminal atribuir as coordenadas das jogadas, matriz 3x3.
